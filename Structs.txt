ESTRUTURA DE DADOS
Uma estrutura de dados é um conjunto de variáveis, geralmente de diferentes
tipos de dados, gerenciado por uma única variável (também denominado 
registro).
Uma estrutura de dados permite, portanto, resolver algumas limitações
inerentes a vetor e matriz onde todos os elementos devem ser do mesmo
tipo de dado.

Declaração de tipo estrutura de dados
Sintaxe:
struct nome_da_estrutura
{
   tipo_elemento nome_elemento1;
   tipo_elemento nome_elemento2;
   tipo_elemento nome_elementen;
};

Pode ser definido da maneira abaixo quando deseja-se declarar o tipo
estrutura de dados e a variável estrutura de dados ao mesmo tempo.
Geralmente os compiladores devolvem um aviso, como se esse procedimento
não fosse uma boa prática

struct
{
   tipo_elemento nome_elemento1;
   tipo_elemento nome_elemento2;
   tipo_elemento nome_elementen;
} nome_da_estrutura;

Declaração da estrutura de dados
struct nome_da_estrutura nome_da_variável_estrutura

Manipular uma estrutura de dados
nome_da_variável_estrutura.nome_elemento

Exemplo:
declaração do tipo estrutura de dados
struct carro
{
    char placa[8], modelo[20];    
    int potencia;
    float valor;
};

declaração da variável estrutura de dados
struct carro terrestre[5];

gets(terrestre[0].placa);
...
gets(terrestre[0].modelo);
...
scanf("%i", &terrestre[0].potencia);
...
